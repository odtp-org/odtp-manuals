# Getting started

This tutorial will guide you through setting up a digital twin and running it

## Start

=== "Dashboard GUI"

    ``` sh
    # go to the project folder
    poetry shell
    ```

=== "Command Line CLI"

    ``` sh
    # go to the project folder
    poetry shell
    odtp dashboard
    ```
